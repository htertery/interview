# Coding Prompt Bank (Starter)

Use these with a 45-60 min solve + 15 min explanation format.

C# requirement for every coding session:

- Use modern C# style compatible with .NET 8+.
- Prefer clear method names, immutable local variables where practical, and guard clauses.
- Explain Big-O complexity and main edge cases before finalizing code.

- COD-001: Given a stream of events, return longest contiguous window satisfying K constraints.
- COD-002: Detect and return all cycle entry points in a directed graph variant.
- COD-003: Design an LRU cache with a secondary eviction rule by priority.
- COD-004: Minimize cost to transform array under operation budget constraints.
- COD-005: Given task dependencies with deadlines, maximize completed tasks.
- COD-006: Build interval query structure supporting updates and max overlap queries.
- COD-007: Find shortest transformation path with weighted edge penalties.
- COD-008: Optimize string rewrite rules with memoization and pruning.
- COD-009: Compute top-K frequent sequences in sliding event windows.
- COD-010: Partition dataset into balanced groups minimizing cross-group cost.
